# Chapter 2: Isaac ROS – Hardware-Accelerated Perception & Localization

This chapter describes how Isaac ROS provides optimized perception and localization capabilities for real-time humanoid robotics using hardware acceleration.

## Learning Outcomes

After completing this chapter, you will be able to:

- Explain what Isaac ROS is and how it extends ROS 2
- Understand hardware acceleration concepts and GPU pipelines
- Describe core capabilities: Visual SLAM (VSLAM), perception pipelines, and sensor integration
- Understand the relationship between Isaac Sim and Isaac ROS
- Explain the data flow from sensors through perception to navigation
- Understand conceptual VSLAM pipeline operations
- Connect sensor data to spatial understanding and navigation

## Chapter Structure

This chapter is organized into the following sections:

1. What Isaac ROS Is and How It Extends ROS 2
2. Hardware Acceleration Concepts (GPU, Pipelines)
3. Core Capabilities: VSLAM, Perception Pipelines, Sensor Integration
4. Relationship Between Isaac Sim and Isaac ROS
5. Data Flow: Sensors → Perception → Navigation
